# Student Marks Streamlit Demo

A simple beginner-friendly **Streamlit web app** that analyzes student marks and displays a clear performance summary.

The app allows a user to enter marks for different subjects, calculates the total marks, percentage, and grade, and displays the result using metrics, a table, and a bar chart.

## Project Overview

This project demonstrates how to build and deploy a small interactive Python web app using **Streamlit**.

It is suitable for:

- Python beginners
- School CS projects
- Streamlit practice
- Basic data visualization demos
- Student performance analysis examples

## Features

- Enter student name
- Input marks for multiple subjects
- Calculate total marks
- Calculate percentage
- Assign grade automatically
- Display result summary using Streamlit metrics
- Show subject-wise marks in a table
- Visualize marks using a bar chart
- Give basic performance feedback

## Tech Stack

| Technology | Purpose |
